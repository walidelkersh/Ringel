# Ringel's Conjecture in Lean 4

A Lean 4 formalization of the Montgomery–Pokrovskiy–Sudakov proof.

The public large-\(n\) theorem is conditional on the explicit source package
CaseABSourceStatement. The deterministic graph theory, Case C construction, proof assembly, and
finite probability infrastructure are documented in the blueprint and audit notes.
